MIXER

MIXER

Mixer

A flexible mixer. You can add multiple channels for mixing by tapping the "+" sign and choosing the signal output icon of the module you want to add to the mix. Make sure you add knobs in the correct order, they cannot be moved afterwards. Each knob will be given the name of the module so if you want to change it, change the module name.

Layers Switch

Layers switch holds a number of parallel racks with only one of them being connected to the output.

Choose which layer is connected by either hitting one of the track buttons or applying a signal between 0.0 and 1.0 on the index input, which will be mapped to the whole range of buttons you have (0.0=first, 1.0=last layer)

  • "CPU save" tells Drambo what to do with layers other than the chosen & connected layer.
    • CPU save = on: Disable processing in all layers but the currently selected one
    • CPU save = off: Keep processing all layers, no matter if selected or not. You may need this in delay or reverb effects for example, when audio needs to be processed in advance to get the desired effect when switching to the respective layer.

X-Fader

Crossfade between inputs. The A/B knob middle position is a 50/50 mix of both signals.

Adder

A super-simple signal mixer that adds all connected signals to one output signal. No adjustments except choosing the signals from modules you want to include. Double-tap on a signal icon to disconnect and remove it.

Layers

Layers module. Holds a number of parallel racks. You may add, remove or rename them. Only one rack is visible at a time, so it's a great tool for simplifying the visual structure. Outputs of all racks are summed at the main module output.

Layers Adder

Layers mixer module holds a number of parallel racks. You may add, remove or rename them. Only one rack is visible at a time, so it's a great tool for simplifying the visual structure.

Layers Mixer

Layers mixer module. Holds a number of parallel racks. You may add, remove or rename them. Only one rack is visible at a time, so its a great tool for making visual structure simpler. Each layer has corresponding volume knob. Outputs of all racks are summed at the main module output.

Layers Crossover

Input signal is divided into 3 frequency bands (configurable via 2 Freq knobs). Each band is processed by a separate layer and mixed down at the end. 2 band split filter slopes are available: 12dB/oct and 24dB/oct. By default, the module mixes down a polyphonic signal to a mono voice (to save CPU). Use the poly option for polyphonic processing.

Pan

Provides stereo panning.

Switch 1-N

Routes the input signal to the selected output. Use the Index input to select input index via external signal (normalized value 0-1).

3 switch modes are available:
• Instant.
• X-Fade - when Index input value is fractional, outputs are cross faded.
• Smooth - transitions are smoothed out.

Switch N-1

Routes one of the input signals to the module's output. Use Index input to select output index via external signal (normalized value 0-1).

3 switch modes are available:
• Instant
• X-Fade - when Index input value is fractional, inputs are cross faded.
• Smooth - transitions are smoothed out.
