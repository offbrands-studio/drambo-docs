SHAPER

SHAPER

Bit redux

Bit depth reduction of the input signal. The signal is quantized to the adjusted bit depth, and the number of quantization steps is equivalent to 2 (# of Bits).

Fur generator

Generates pulse wave that fits zero crossings or peaks of incoming signal.

Decimator

Resample the input signal with an adjustable sample rate. (aliasing-supressed version)

Graphic shaper

A Waveshaper that lets you draw any shaping curve. Can shape any kind of CV signal.

Tip: Use it after a SAW wave oscillator with "Anti-aliasing"=Off to draw your own phantasy waveform.

Shaper

A multimode waveshaper with different shaping presets. 3 saturation modes and 2 folding modes are available. Bias greater than zero turns on DC offset blocker.

Pulse divider

Pulse Divider module divides the frequency of the incoming pulses by integer ratios.

S&H

Sample and Hold: With every gate impulse, store the current value of the input signal and send it to the module output. The output remains constant until the next gate signal triggers sampling of the input value again.

Slew limiter

Delays the rising or falling of a signal by sweeping from one value to the other.
