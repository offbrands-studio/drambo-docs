MIDI

MIDI

CC modulator

Convert incoming MIDI CC events into a modulation CV signal. Press the Learn button and move a MIDI controller to assign a CC number automatically.

Monitor

A MIDI message monitor. Can read and display incoming messages as well as messages from internal MIDI modules.

Note: Not all messages are shown in order to keep the display readable.

MIDI input

Provides a dedicated MIDI input from the selected MIDI port.

  • "Channel" lets you choose which MIDI channel to receive from.

  • "Omni" means: Receive MIDI from all channels 1 to 16.

MIDI output

Send incoming MIDI to an external MIDI output.

MIDI to CV

Turn incoming MIDI data into corresponding CV signals: Gate, pitch and velocity.

CC generator

Turns internal CV signal into midi CC message to be used to control or modulate external AUv3, IAA or hardware.

Select the CC number from the drop down menu and route its output to a Midi Output module.

CC generator (multi)

Use knobs to generate many MIDI CC messages and mix them into incoming MIDI stream. Knobs are modulatable. Double tap on a knob to set CC number.

Note generator

Turns internal Note - gate, velocity, pitchsignals into midi Note message, so various CV generator and sequencer modules can be used to drive external AUv3, IAA or hardware. Route its output to Midi Output module.

Sysex generator

No module help yet. Come back later please :)

Arp

Arp allows the player to automatically step through a sequence of notes based on the player's input, thus creating an arpeggio. Its a MIDI effect plugin, so use it with MIDI signals (put the module before MIDI 2 CV or Instrument rack in MIDI mode).

• Rate, Sync - set playback frequency, tempo synchronization mode (modulatable).
• Gate - Set gate length (modulatable).
• Host sync - When active, generated MIDI events are synchronized with host tempo, in other case sequence starts as soon as you press the first key.

• Range - Number of octaves, the sequence goes through.
• Mode - up, up and down, down, random, as play. Defines the direction of the sequence.

Optional gate input: When connected, the internal ARP sequencer is triggered only on gate signal.

Euclidean sequencer

Euclidean sequencer: Divide a number of beat subdivisions over a time interval and place MIDI notes as evenly as possible into the sequence. Generated events are mixed into an incoming MIDI stream, so this module may be chained to generate different notes on the same track. Use the Time swing module before for beat shuffling.

Channel filter

Depending on type setting, it passes only MIDI note/cc messages on specific channel (Filter) or sets a fixed channel to all incoming MIDI messages (Set). Tip: Use two of these modules to build MIDI channel based routings.

Chord

Create a chord by adding additional voices to incoming midi Note(s). Use knobs to define how each additional voice should be shifted in semitones) from incoming midi Note(s).

Chords with up to 6 voices can be created.
Keeping a knob at its default position "0" will not add another voice to the chord.

Latch

Latches incoming MIDI notes when Latch button is active. Works in 2 modes:

  • replace - replaces all latched notes each time a new note is coming
  • toggle - incoming notes are added to existing latched notes pool.

Chance

Add probability to an incoming midi Note. Enable Chord mode to add probability to each note of an incoming chord.

Delay

Signal delay, modulatable and adjustable between 0.0ms and 1.0s delay time.

Humanizer

Varies note values like velocity and note length (gate) and can push notes back with random offset.

MIDI Mixer

Merges MIDI signals from different MIDI sources.

MIDI Mixer (mutes)

Combines multiple MIDI inputs into a single output. Each input may be muted.

MIDI switch 1-N

A MIDI-aware switch to route one input to one of multiple outputs. Will take care of correctly handling MIDI Note messages (i.e. sending Note Off messages when a MIDI port with sustained notes is switched away from).

Note filter

Use drop down menus to define a range of notes, where From will become the lowest and To will become the highest note of the range.
All notes outside of the range will be filtered out.

MIDI to Poly

A polyphonic MIDI voice distributor. The incoming MIDI signal is divided into voices and sent to 4 MIDI outputs, each representing a separate voice.

Pitch bend

Generates MIDI pitch bend message.

Retrig

Retriggers incoming MIDI Note messages. You can set rate, beat division and velocity.

Strum

Play a chord split into its individual notes. Requires that all chord notes start (and optionally end) at the same time.

Transpose

Transpose incoming midi Note(s) by Amount * Interval (in semitones).
A quick and effortless way to transpose musical phrases up and down when turning patterns into songs. A midi 'guide track' of sorts...
