MATH

MATH

Add

Add two input signals.

Divide

Divide the signal value of Input 1 by the signal value of Input 2. Negative numbers are supported. Divide by zero is eliminated and will return a very high result instead.

Function

Offers a choice of preset operations on the two input channels.

• A+B: Add the values of A and B
• A-B: Subtract B from A.
• A*B: Multiply A by B.
• Maximum: Always output the higher one of the two input values.
• Minimum: Always output the lower one of the two input values.
• A>B: Outputs 1 if A is higher than B, otherwise O • B>A: Outputs 1 if B is higher than A, otherwise 0

Maximum

Find and forward the higher value of two input signals.

Minimum

Find and forward the lower value of two input signals.

Multiply

Multiplies input values.

Subtract

Calculate the difference between the two input signals.

1/x

Calculates the reciprocal (multiplicative inverse) of the input value.

Clip

Clips the input signal to Low and High range.

Cosinus

Calculates exponential function of the input signal.

Exp

Mirrors the lower half of the input signal (below the waveform zero line) to the upper half.
Derived from an electronic bridge rectifier with four diodes. Creates even more overtones than the Half Rectifier.

Full rectify

Cuts away the lower half of the input signal (below the waveform zero line). Derived from a simple electronic rectifier with one diode only. Adds overtones to the signal.

Half rectify

Cuts away the lower half of the input signal (below the waveform zero line). Derived from a simple electronic rectifier with one diode only. Adds overtones to the signal.

Integrator

Calculates continous integral of incoming signal.

Differentiator

Sends a difference between current and previous signal value.

Log

Calculate the logarithmic of the input signal. Useful for compressing the lower range and expanding the upper range of values.
Expects a positive signal at the input. Input values at or below zero will result in Output=zero.
If you want to make use of the full scale of a bipolar input signal, make sure you add an appropriate offset to it before feeding it this module.

Number

A signal generator that provides a pre-set DC signal. The entered value can be positive or negative.

Offset

Adds a constant value to the input signal.

Quantize

Round incoming signal to the nearest integer value. (e.g. 1,2 -> 1). Use the Scale or Scale&Offset modules to adapt it to the signal level.

Scale

A signal scaler that can boost or attenuate a signal. Range is the preamp gain, it can also be a negative number like -0.34.

Sinus

Calculate the sine function of the input signal.

Sqrt

Calculates the square root of the input signal. Input values at or below zero always result in zero. If you want to make use of the full scale of a bipolar input signal, make sure you add a positive offset to the input signal before connecting it to this module.

AND

Logical AND gate. Output is 1 only when both inputs are greater than zero.

OR

Logical OR gate. Output is 1 only when one of inputs is greater than zero.

XOR

Logical eXclusive-OR gate. Output is 1 if only one of the inputs is 1. If both inputs have the same logical level then the output is zero.

NEG

Logical negation. All values equal or below 0 turn into 1. Greater than 0 turn into 1.
