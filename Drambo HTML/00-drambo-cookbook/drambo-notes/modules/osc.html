OSC

Oscillator (mini)

Similar to main Oscillator. Has additional Sine Nine and Sine PD shapes. Pulse Width knob also controls phase distortion amount in Sine PD shape. Ideal FM modulator.

Oscillator

A standard subtractive-style oscillator with a variety of waveforms and modulation inputs for pitch and pulse width. Pulse width adjustment is only available when the Pulse waveform is selected. Tip: Add one of the pitch modules before for more precise pitch adjustment.

Wavetable oscillator

an ultra-clean wavetable oscillator.

Features:

Support for Serum wavetable format (WT frames each with 2048 samples length), up to 256 waves in a single oscillator, morphing, 7 voices stereo unison, modulatable Position, audio-rate modulatable Phase and Tune, modular wavetable manipulation.

The Wavetable oscillator contains an internal rack for advanced wavetable effects, using additional modules like Wave effect and Wave filter. Tap on the rack icon in the bottom-right corner to open the internal rack that is used for additional wavetable processing.

Use the Wave effect and Wave filter modules from PROCESSOR / Spectral section.

41 great wavetables created by Echo Sound Works included.

You can also import standard wave files that are not dedicated wavetables (and therefore won't scan as smoothlv).

Oscillator: Supersaw

A perfect emulation of the famous JP8K supersaw oscillator. (7 detuned saw waves)

FM operator

The basic element (operator) of FM synthesis.
An internal oscillator's frequency or phase is linearly modulated by FM input (autoconnected). Internal feedback changes oscillator waveshape from square wave through sine to saw. Frequency ratio may be set to zero. In this case the FM Operator "is stopped" and acts like a sophisticated waveshaper / through-zero oscillator.

  • FM Amount - controls how much modulator (connected to FM input) affects FM modulation.
  • Ratio - base frequency ratio
  • Detune - base frequency detune
  • Mod - Phase / Freq - switch between phase or linear frequency modulation
  • Feedback - amount of output signal that is internally fed back to FM input. Changes waveshape: square-sine-saw
  • Attack, Decay - set attack and decay length of internal amp envelope
  • Retrig - Determines if oscillator should restart phase on each trigger
  • Velocity - amount of velocity influence on amp envelope level
    End level - bottom level of amplitude envelope generator
    Combine several FM Operators in serries or using mixer to achieve different FM structures.

Noise

Noise generator. Allows for choosing between "White" and "Pink" noise which has less high frequency content. Switch to stereo for uncorrelated stereo noise. Polyphony: If you want to have separate noise source per voice, connect gate input to polyphonic gate output.

Impulse

Create a short impulse from Gate and Velocity inputs. Impulse shape depends on the Shape setting and the impulse length can be adjusted between super-narrow and up to 50ms. Noise allows you to add noise with the same envelope as the impulse and Tone is a simple 6dB low pass filter to further shape the signal. Useful but not limited to creating excitation impulses for the Modal Resonator, Comb Filter and others.

AN Kick

A physically-informed digital model of famous TR808 kick generator cirucit.

Formant oscillator

Sunthesizes and approximates the human voice by generating vowel sounds (and more).

These can be manipulated to provide deep low baritones or high pitched squeaks.

Use Formant parameter to morph between 5 vowels, that you set use a row of select controls on the right side.
