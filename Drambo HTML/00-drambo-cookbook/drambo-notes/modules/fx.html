FX

FX

Buffer OSC

On each incoming trigger, a portion of the input signal is sampled to a buffer and played in a loop.
Loop length is determined by the Pitch parameter.

Buffer Rescan

On each synchronized cycle (defined by Size param), the incoming signal is recorded and replayed with defined Speed (bidirectional)

Buffers are crossfaded to prevent clicks.

Buffer Repeat

Loop and repeat incoming signal. Buffer size is tempo synchronized.

Buffer stop

Simulates tape start / stop effect.

note: tape start is crossfaded with original signal. It's not possible to make tape start for realtime signal without lookahead or latency

Channel EQ ①

DJ style EQ with fixed curves and frequencies.

• Low - low shelf @100Hz
• Mid - peak @ 1.5kHz
• High - high shelf @10kHz

Chorus ①

A Classic Chorus effect that can be run in either mono or stereo configuration. The effect signal is always mixed with the input signal, an effect-only signal is not available.

Compressor ①

A classic dynamics compressor with lookahead and RMS mode.

Delay

Signal delay, modulatable and adjustable between 0.0ms and 1.0s delay time.

Delay FX ①

An audio delay that can either be freely adjusted between 20ms and 2.5s or synchronized to adjustable beat divisions (based on 3/4 and 4/4 beats).

Dry/Wet adjusts the original vs effect levels. The rightmost setting will emit the delayed effect signal only.

Delay rack ①

A Delay with an internal rack that is used to process the signal in the feedback loop.

Frequency shifter

Shifts all the frequencies in the input signal up or down by a certain amount.

This kind of shifting will ruin the harmonic footprint of the input signal, making it sound dissonant.

• Amount - linear amount of frequency shift from (-Range to Range), zero = no shift

• Range - exponential range in Hz in which Amount param works.

The output shift is: Amount * exponential (Range).

Tips:

  • If you want to modulate shift exponentially then modulate the Range knob.

  • If you want to modulate shift in a linear manner then modulate the Amount knob.

  • If you want shift by tracking the current keyboard frequency then modulate the Range param with keyboard frequency (= Pitch signal)

Noise gate

Noise gate is a specific case of the dynamics processor.

It removes parts of the signal with levels below the selected threshold.

Signal is smoothed by attack, hold, release envelope.
Envelope detector signal is available at additional output: SG env.

You may use it for further processing, e.g. control filter cutoff.

Use sidechain input to use other signal for envelope detection.

Phaser ①

A classic 6 stages phaser with feedback. (zerodelay technique)

Flanger ①

A flanger works by mixing two identical audio signals together, with one of the signals playing at a slightly slower speed.

This creates the effect of two tape recordings playing simultaneously, but with one tape player going slightly slower than the other.

Pitch shifter ①

The pitch shifter changes the pitch of an incoming audio signal. It adds some latency.

Convolver ①

A simple but very useful convolution (IR) processor. Multiplies the spectrum of the input signal with the spectrum of the loaded impulse response. Can be used for EQ, delays, reverbs etc. but cannot replicate amplitude dependent characteristics (to add these, use the Graphic shaper).

Reverb (FDN) ①

An algorithmic reverb audio effect based on a delay feedback network. Lush and precise.

Reverb (Plate) ①

An algorithmic reverb audio effect based on a plate model.

Stereo width

Reduces the stereo signal width (i.e. L/R channel differences) by an adjustable amount.

If one channel is the perfectly phase-reversed (inverted, negated) version of the other channel, setting Width to zero will result in no sound output.

Punch-in looper

This module loops incoming audio signal according to incoming MIDI notes, that represent step position in a bar.

Use it on a send track, e.g. Master.

Punch-in FX

Punch-in FX module is a set of effect layers, each activated by incoming MIDI note (piano key).

Use it on a send track, e.g. Master.
